agents.models#

class agents.models.VisionModel#

Bases: agents.models.Model

Object Detection Model with Optional Tracking.

This vision model provides a flexible framework for object detection and tracking using the mmdet framework. It can be used as a standalone detector or as a tracker to follow detected objects over time. It can be initizaled with any checkpoint available in the mmdet framework.

Parameters:

  • name (str) ‚Äì An arbitrary name given to the model.

  • checkpoint (str) ‚Äì The name of the pre-trained model‚Äôs checkpoint. All available checkpoints in the mmdet framework. Default is ‚Äúdino-4scale_r50_8xb2-12e_coco‚Äù.

  • cache_dir (str) ‚Äì The directory where downloaded models are cached. Default is ‚Äòmmdet‚Äô.

  • setup_trackers (bool) ‚Äì Whether to set up trackers using norfair or not. Default is False.

  • tracking_distance_function (str) ‚Äì The function used to calculate the distance between detected objects. This can be any distance metric string available in scipy.spatial.distance.cdist Default is ‚Äúeuclidean‚Äù.

  • tracking_distance_threshold (int) ‚Äì The threshold for determining whether two object in consecutive frames are considered close enough to be considered the same object. Default is 30, with a minimum value of 1.

  • deploy_tensorrt (bool) ‚Äì Deploy the vision model using NVIDIA TensorRT. To utilize this feature with roboml, checkout the instructions here. Default is False.

  • _num_trackers (int) ‚Äì The number of trackers to use. This number depends on the number of inputs image streams being given to the component. It is set automatically if setup_trackers is True.

  • init_timeout (int, optional) ‚Äì The timeout in seconds for the initialization process. Defaults to None.

Example usage:

model = DetectionModel(name='detection1', setup_trackers=True, num_trackers=1, tracking_distance_threshold=20)  # Initialize the model for tracking one object
